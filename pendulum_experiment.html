单摆原理

单摆是由一根不可伸长的细线悬挂一个质点构成的系统。当摆角较小时，单摆做简谐运动。

单摆的周期公式为：

T = 2π√(L/g)

其中，T是周期，L是摆长，g是重力加速度。

单摆运动的特点：

  • 摆长越长，周期越长
  • 重力加速度越大，周期越短
  • 在小角度范围内，周期与摆角无关

单摆的运动方程：

d²θ/dt² + (g/L)sinθ = 0

当角度较小时，可以近似为：

d²θ/dt² + (g/L)θ = 0

这是典型的简谐运动方程。
